moons/debouncer~ Debouncer

new Debouncer(spec)

General use event debouncing constructor. Debouncing is done off of the requestAnimationFrame method
Parameters:
Name Type Description
spec Object Initialization object
Properties
Name Type Argument Default Description
element Element The element to which the listener is bound
elementWithValue Element <optional>
 spec.element The element with the changing value. This is common when binding to 'scroll'.
event string The event to bind to
value string The property name on elementWithValue that provides the updating value

Example
var scroller = new Debouncer({
  element: window,
  elementWithValue: document.body,
  event: 'scroll',
  value: 'scrollTop'
});
